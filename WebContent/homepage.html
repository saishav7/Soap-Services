Following are the parameters involved in the web services:-

NAME TYPE DESCRIPTION EXAMPLE
sec String The security code of the financial instrument (i.e., stock). All data rows in the Market Data file having a security code that matches this parameter should be included in the output Market Data file. WSSS
startDate dateTime The earliest timestamp (i.e., the first line) in the Market Data file associated with the eventSetID. Rows that have a timestamp on or after (i.e., inclusive) this parameter and also satisfy the condition for endDate, are included in the output Market Data file 2002-04-08T16:59:00.000
endDate dateTime The latest timestamp (i.e., the last line) in the Market Data file associated with the eventSetID. Rows that have a timestamp on or before (i.e., inclusive) this parameter and also satisfy the condition for startDate, are included in the output Market Data file 2002-04-09T17:48:00.000
dataSource anyURI This is a http:// URL that points to the input Market Data file http://adage.cse.unsw.edu.au:8080/MarketData03.csv
eventSetId String An opaque "handle" or reference that refers to the output Market Data file (also called an event set: because it stores an ordered set of events) tsb-9b7e45ad
targetCurrency String It is the currency to which the market-data file is to be converted USD

Import and Download

This service enables customers to import market-data files, which is done by filtering the data file using four parameters namely sec, startDate, endDate and dataSourceURL. On clicking submit, the output file will be created and an eventSetId will be generated, which is essentially a reference to the required file. The file call be downloaded by clicking on the download button.

Click here for more details about the parameters used.

Market Data Summary

This service allows customers to get details of a market-data file. It expects an eventTypeId as input and gives the details of the file which includes eventSetId, startDate, endDate, marketType, currencyCode and fileSize.

Currency Converter

This services allows customers to convert the currency of the market-data files to the desired type(USD, GBP, EUR etc). Note that the default existing currency is AUD. The service creates a new file containing the converted currency values. An eventSetId is returned, which is a reference to the output file.
